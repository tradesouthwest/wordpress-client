== Description ==
Business Directory Plugin addon to make a listing viewable only to assigned user.
Opens under Settings > TSW Views Addon

== Installation ==

This section describes how to install the plugin and get it working.
1. Upload `views-addon` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
